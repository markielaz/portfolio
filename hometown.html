'My Hometown' - putting HTML and CSS into practice

When I started the Northcoders bootcamp, I already had experience in web development and was more than comfortable using HTML and CSS to build basic sites.

During the Fundamentals section of our course, we were encouraged to create an example website for portfolio use, based on our hometown.

This is the site I ended up creating. I had fun trying to make it a little more interesting than the usual traditional layout. It was important to me that the site would function responsively and be easy to navigate. It also needed to be accessible.

I intended to treat it as a sandbox environment to some degree, where I could go back and add more features as they occur to me - although to be honest, I suspect I would now rather just continue making more apps instead!
